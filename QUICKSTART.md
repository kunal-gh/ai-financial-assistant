# Quick Start Guide

Get started with AI Financial Assistant in 5 minutes!

## For Users

### 1. Download & Install (2 minutes)

1. Download `app-release.apk` from [Releases](../../releases)
2. Install on your Android device
3. Grant microphone permission

### 2. First Launch (1 minute)

- See "Created by SW" splash screen
- App opens to Dashboard
- Sample data is pre-loaded for testing

### 3. Try Features (2 minutes)

## For Developers

### 1. Clone & Install (3 minutes)

```bash
git clone https://github.com/yourusername/ai-financial-assistant.git
cd ai-financial-assistant
npm install
```

### 2. Run (2 minutes)

```bash
# Terminal 1: Start Metro
npm start

# Terminal 2: Run Android
npm run android
```

## Troubleshooting

**Voice not working?**
- Check microphone permission
- Test on real device (not emulator)

**App crashes?**
- Reinstall the app
- Clear app data

**Need help?**
- Check [docs/](docs/)
